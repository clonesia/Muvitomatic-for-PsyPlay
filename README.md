# Muvitomatic for PsyPlay Themes
### Main Features
- Generate Movie from ImDB or Gdriveplayer.us
- Post All Movies
- Post All TV Show
- Post All Series / Anime
- Schedule Post Movie / Series / Anime
- Generate Movie / Series / Anime General Tags (eg. Download %postname% Full Movie )
